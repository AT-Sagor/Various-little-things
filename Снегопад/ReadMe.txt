Version 1.1.0

- Добавлена рандомизация размера снежинки.
- Уменьшено размытие снега.
- Введена скорость падения снега в зависимости от размера - чем дальше, тем медленнее.
- Введена общая переменная для количества снежинок.
- Увеличено количество снежинок со 100 до 200.

Version 1.0.0

Обыкновенные размытые блоки имитирующие снежинки. Начинают в произвольных местах над верхним краем контейнера, а потом медленно падают вниз. При прохождении определённой дистанции снова располагаются в случайной позицией над экраном и так по замкнутому циклу.
