Version 1.3.2

Добавлен плавный переход для избранных цветов и их фона.

Version 1.3.1

Код избранных цветов был оптимизирован.

Version 1.3.0

Можно добавить до трёх цветов в избранное.

Функции избранных цветов:

 - Левая кнопка мыши: добавить цвет в избранное;
 - Повторное нажатие левой кнопки мыши: убрать цвет из избранного, но в поле ввода появиться его код и body окраситься в этот цвет.
 - CTRL + левая кнопка мыши: добавить фон избранным цветам, как у указанного избранного цвета.
 - ALT + левая кнопка мыши: очистить фон для избранных цветов.

Version 1.2.0

При открытии страницы body генерирует случайный цвет с прозрачностью.
Присутствует поле для ввода своего цвета. Скрипт распознает любую разрешённое значение для "background".
Есть кнопка, которая генерирует случайный цвет с прозрачностью и записывает его код в строку ввода, чтобы моно было потом его копировать для своих нужд.
