Version 1.2.0

При открытии страницы body генерирует случайный цвет с прозрачностью.
Присутствует поле для ввода своего цвета. Скрипт распознает любую разрешённое значение для "background".
Есть кнопка, которая генерирует случайный цвет с прозрачностью и записывает его код в строку ввода, чтобы моно было потом его копировать для своих нужд.
